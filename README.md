shell script for making a MISP backup based on 
https://github.com/daverstephens/The-SOC-Shop/blob/master/MISP-Stuff/misp-backup.sh

Installation
============

Modify config file:
````
cp misp-backup.conf.sample misp-backup.conf
#adjust values
vi misp-backup.conf
````

Running
=======

run the script:
````
sh misp-backup.sh
````
